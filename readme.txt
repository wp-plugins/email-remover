This is a simple plugin that removes all emails from an wordpress rendered page

== Description ==

This plugin captures entire wordpress page and removes all email links from it. It also replaces the email address @ and . with at and dot.

== Installation ==

1. Upload `email-remover.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
